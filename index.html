    <h2>1. Procesamiento de Imágenes por IA (Gemini & Nano Banana)</h2>

    <p>Nuestra aplicación utiliza la API de <strong>Gemini y Nano Banana 2 (Google)</strong> para generar y editar contenido visual basado en las fotos que usted proporciona.</p>

    <ul>

        <li><strong>Uso de Fotos:</strong> Las imágenes cargadas se envían de forma segura a los servidores de Google para su procesamiento técnico inmediato.</li>

        <li><strong>Privacidad:</strong> No utilizamos sus fotos para entrenar modelos de IA ni para fines publicitarios.</li>

        <li><strong>Retención:</strong> Las imágenes temporales se eliminan de nuestros sistemas una vez generado el resultado solicitado.</li>

    </ul>
